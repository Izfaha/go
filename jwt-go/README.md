# JWT - golang

## Pre-requisites

| module         | command                                                                                                    |
| :------------- | :--------------------------------------------------------------------------------------------------------- |
| gorm           | go get -u gorm.io/gorm                                                                                     |
| postgres       | go get -u gorm.io/driver/postgres                                                                          |
| gin            | go get -u github.com/gin-gonic/gin                                                                         |
| bcrypt         | go get -u golang.org/x/crypto/bcrypt                                                                       |
| godotenv       | go get -u github.com/joho/godotenv                                                                         |
| jwt            | go get -u github.com/golang-jwt/jwt/v5                                                                     |
| compile daemon | go get -u github.com/githubnemo/CompileDaemon<br>and go install github.com/githubnemo/CompileDaemon@latest |
